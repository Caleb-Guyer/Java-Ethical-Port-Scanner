# Java Ethical Port Scanner

A lightweight Java command-line tool that scans for open ports on a target host. 

**Purpose:** Educational project to explore networking, sockets, and basic cybersecurity (port scanning / reconnaissance).

### Features
- User-defined target host and port range
- Timeout handling to prevent hanging on closed ports
- Clean output showing only open ports
- Built with pure Java (no external libraries)

**Ethical Use Only**  
Only scan systems you own or have explicit permission to test.
